## Setting up LimsPlus Portal

### Prerequisites:

- [Node](https://nodejs.org/en/) or
- [Yarn](https://yarnpkg.com/lang/en/)

## Technology use
 - 1. React + NextJs
 - 2. Tailwindcss
 - 3. Node, NestJs, Graphql, Typeorm, Mongodb
 - 4. Mobx (State management)
 
 

```sh
git clone https://github.com/appasaheb4/limsplus-portal
cd limsplus-portal
yarn install
npm run dev
```
